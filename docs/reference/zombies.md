# PvZ1 Zombie Catalogue

Comprehensive reference of all zombie types in Plants vs. Zombies (original/GOTY edition). Stats reflect the original PC/console release.

## Legend

- **Health** — Relative hit points. Normal = 181 (10 pea hits to kill). Values shown as multiples of Normal where helpful.
- **Speed** — Movement category: Slow, Basic, Fast, Variable (changes mid-behaviour)
- **Special-ability** — Unique mechanic or behaviour beyond walking and eating
- **First-appears** — Level at which the zombie is first encountered in Adventure mode
- **Counters** — Plants or strategies that are particularly effective
